Create React App 시작하기
이 프로젝트는 Create React App을 사용해 생성되었습니다.

사용 가능한 스크립트
프로젝트 디렉터리에서 아래 명령어를 실행할 수 있습니다.
npm start
개발 모드로 앱을 실행합니다.
브라우저에서 http://localhost:3000 을 열면 확인할 수 있습니다.
코드를 수정하면 페이지가 자동으로 새로고침됩니다.
콘솔에는 lint 오류가 표시될 수 있습니다.

npm test
테스트 러너를 인터랙티브 감시 모드로 실행합니다.
자세한 내용은 테스트 실행 관련 문서를 참고하세요.

npm run build
앱을 프로덕션용으로 build 폴더에 빌드합니다.
React가 프로덕션 모드로 최적화되어 최고의 성능을 낼 수 있도록 번들링됩니다.
빌드 결과는 최소화(minify)되며, 파일 이름에 해시값이 포함됩니다.
이제 배포할 준비가 완료된 상태입니다.
배포에 대한 자세한 내용은 관련 문서를 참고하세요.

npm run eject
⚠️ 주의: 이 작업은 되돌릴 수 없습니다.
기본 빌드 설정이 마음에 들지 않을 경우 언제든지 eject 할 수 있습니다.
이 명령을 실행하면 하나로 묶여 있던 설정이 프로젝트 내부로 모두 복사됩니다.
이후에는 webpack, Babel, ESLint 등의 설정을 직접 수정할 수 있습니다.
eject 이후에도 기존 명령어들은 계속 사용 가능하지만, 복사된 설정을 기반으로 동작합니다.
꼭 eject를 사용할 필요는 없으며, 기본 설정만으로도 중소 규모 프로젝트에는 충분합니다.
